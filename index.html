Jackson XML databind by FasterXML

Extension for Jackson JSON processor that adds support for serializing POJOs as XML (and deserializing from XML) as an alternative to JSON

Dependencies

(see pom.xml for details)

  • Jackson (1.7.0 or above); 'core' and 'mapper' modules, 'xc' optional (to use JAXB annotations)
  • Stax XML parser like Woodstox or Aalto
  • Stax2 API (needed by Stax parser)
